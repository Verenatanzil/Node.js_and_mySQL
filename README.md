# BAMAZON

Bamazon is a CLI App. This application let the user see the list of available products that are based in mySQL. With this app the user then will be asked to enter item ID and the amount of item that they would like to buy. Afterward, the total cost will be calculated and the database will be updated. User then will be asked if they want to continue to shop or stop.
